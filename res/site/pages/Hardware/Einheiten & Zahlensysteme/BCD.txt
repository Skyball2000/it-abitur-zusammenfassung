# 2. BCD
> Binary Coded Decimal

Binary Coded Decimal ist eine Möglichkeit, jeweils eine Dezimalstelle als Binärzahlen darzustellen. Hierbei werden vier Bits verwendet:<br><br>

`0000b` für `0d`<br>
bis<br>
`1001b` für `9d`<br><br>

So können auch lange Dezimalzahlen als eine Folge von BCD-Codes dargestellt werden:<br>
`4285` → `0100 0010 1000 0101`<br><br>

Ein Anwendungsfall wären z.B. 7-Segment-Anzeigen
